# PlaylistMaker
Приложение помогает создавать плейлисты треков из музыкальной базы iTunes, применимо в сфере медиа и развлечений.

## Особенности
- сохранение одного трека в разные плейлисты;
- офлайн доступ к информации об избранных треках;
- кэширование последних запросов;
- добавление пользовательских изображений для плейлистов;

## Достижения
Оптимизировал загрузку треков из плейлиста с помощью SQL-конструкции JOIN вместо поиска по идентификаторам, что ускорило получение треков на 31% и отображение контента на экране

## Технологии и архитектура

### Архитектура

**Архитектурный паттерн** - MVVM  
**DI** - Koin  

### Навигация и интерфейс
**Навигация** - Jetpack Navigation  
**Подход**: Single Activity, Fragments  
**Безопасный доступ к элементам интерфейса** - ViewBinding  
**Примеры использованныех элементов интерфейса**: RecyclerView, ViewPager2, BottomSheet  

### Слой представления
- ViewModel  
- LiveData  

### Асинхронность
Kotlin Coroutines & Coroutines Flow

### Работа с iTunes API
**Сетевые запросы** - Retrofit2  
**Сериализация JSON** - Gson  
**Загрузка и отображение изображений** - Glide  
 
### Хранение данных
**Сохранение избранных треков, плейлистов** - Room  
**Сохранение настроек пользователя (выбор темы, последние открытые треки)** - SharedPreferences  
**Сохранение пользовательских обложек плейлистов** - Internal Storage  
 
### Воспроизведение 
**Плеер для превью треков** - MediaPlayer
